## 1. 프로젝트 개요

도난당한 루브르 박물관 유물에서 영감받은 **인터랙티브 3D 웹 전시관**.

## 2. 이전 세션에서 변경된 사항 (v1 → v2)

### 삭제된 기능
- **MediaPipe 손 인식** — CDN, 웹캠, 제스처 로직 전부 삭제
- **퍼즐 미니게임** — HTML, CSS, JS 전부 삭제 (게임용 Three.js 씬 포함)
- **사이드바 레이아웃** — 왼쪽 사이드바 UI 제거
- **배경 이미지** — `Galerie_d'Apollon_(Louvre).jpg` 오버레이 제거

### 추가/변경된 기능
1. **풀스크린 히어로 레이아웃** — 3D 유물이 화면 전체를 차지
2. **← → 슬라이드 전환** — 텍스트 마스크 애니메이션 (translateY 110%)
3. **"Continue Reading" → 상세 뷰** — 유물이 왼쪽으로 이동, 오른쪽에 에디토리얼 타이포
4. **커스텀 커서** — 흰색 원이 lerp(0.12)로 부드럽게 따라다님, 버튼 위에서 확대
5. **로딩 화면** — "LOUVRE" 골드(#c4a55a) 텍스트 펄스 애니메이션
6. **카메라 lerp 시스템** — 모드 전환 시 카메라가 부드럽게 이동 (camGoal 객체)
7. **드래그 스크롤** — 상세 뷰에서 마우스 드래그로 콘텐츠 스크롤

### 디자인 방향
- **교수님 피드백**: "게임 같지 않게, 명품 브랜드 카탈로그 느낌으로"
- 참고 레퍼런스: 샤넬 제품 페이지, Egyptian Dynasties Transitions (Dribbble), Alexander McQueen 웹사이트
- 다크 배경(#0c0c0c) + 화이트 타이포 + 미니멀 UI

## 3. 현재 완성된 기능

### 3-1. 히어로 모드 (메인 화면)
- Three.js 3D 뷰어 — 유물이 화면 가운데 크게
- 좌하단: 카테고리(CROWN 등) + 큰 제목(58px) + 서브타이틀 + "Continue Reading" 버튼
- 좌우 화살표(← →): 클릭 시 텍스트 마스크 슬라이드 + 캔버스 페이드 전환

- "BACK" 버튼 좌상단 (이탤릭)
- "Scroll Down" 우하단

## 6. 주의사항 (과거에 문제됐던 것들)

| 이슈 | 해결 |
|------|------|
| 3D 모델 어두움 | `THREE.NoToneMapping` + ambient 2.5 유지 |
| 에메랄드 목걸이 보석 안 보임 | `rotationX: Math.PI/2` 필수 |
| GLB 로드 실패 시 무한 로딩 | 에러 콜백에서도 로딩화면 숨김 |
| 스파클/파티클 이펙트 | 사용자가 싫어함 — 넣지 말 것 |
| 교수님: "게임 같다" | 명품 카탈로그 스타일로 리디자인 완료 |
| 유물 오른쪽 치우침 | 카메라 x값 양수 = 유물이 왼쪽에 보임 (부호 주의!) |

## 7. 카메라 설정 참고

```javascript
// 히어로 모드 (가운데)
heroCamera = { x: 0, y: 1, z: 12, tx: 0, ty: 0.5 }

// 상세 뷰 (유물 왼쪽으로)
detailCamera = { x: 4, y: 1, z: 7, tx: 4, ty: 0.5 }
// x를 양수로 하면 카메라가 오른쪽으로 → 유물이 왼쪽에 보임
```
